Условный оператор if

Оператор if(...) вычисляет условие в скобках и, если результат true, то выполняет блок кода.

Может содержать необязательный блок «else» («иначе»). Выполняется, когда условие ложно.

Иногда, нужно проверить несколько вариантов условия. Для этого используется блок else if.

Подробнее

Условный оператор '?'

Работает аналогично if...else, но используют, чтобы присвоить переменной то или иное значение в зависимости от условия.

Не рекомендуется заменять конструкции if...else, где используются сложные действия.

Конструкция "switch"

Конструкция switch заменяет собой сразу несколько if. Она представляет собой более наглядный способ сравнить выражение сразу с несколькими вариантами.

Методы для массивов

.length Длина массива (это свойство, не метод)
.pop() Извлекает элемент с конца
.push (...items) Добавляет элементы в конец
.shift() Извлекает элемент с начала
.unshift(...items) Добавляет элементы в начало
.slice(start, end) Возвращает новый массив, в который копирует элементы, начиная с индекса start и до end (не включая)
.splice(pos, deleteCount, ...items) Начиная с индекса pos, удаляет deleteCount элементов и вставляет items
.concat() Создаёт новый массив, в который копирует данные из других массивов
.indexOf(item, pos) Ищет item, начиная с позиции pos, и возвращает его индекс или -1, если ничего не найдено
.lastIndexOf(item, pos) Ищет item, начиная с позиции pos, и возвращает его индекс или -1, если ничего не найдено. Ищет с конца
.includes(value) Возвращает true, если в массиве имеется элемент value, в противном случае false
.filter(func) Фильтрует элементы через функцию и отдаёт массив значений, при прохождении которых через функцию возвращается true
.find(func) Ищет элементы через функцию и отдаёт первое значение, при прохождении которых через функцию возвращается true
.findIndex() Похож на find, но возвращает индекс вместо значения
.forEach(func) Вызывает func для каждого элемента. Ничего не возвращает
.map(func) Создаёт новый массив из результатов вызова func для каждого элемента
.sort(func) Сортирует массив «на месте», а потом возвращает его
.reverse() Меняет порядок следования элементов на противоположный «на месте» и возвращает изменённый массив
.split() Преобразует строку в массив
.join() Преобразует массив в строку
.reduce(func, initial) Вычисляет одно значение на основе всего массива, вызывая func для каждого элемента и передавая промежуточный результат между вызовами
Array.isArray(arr) Проверяет, является ли arr массивом
.every(func) Проверяет, удовлетворяют ли все элементы массива условию, заданному в передаваемой функции
.some(func) Проверяет, удовлетворяют ли хоть один элемент массива условию, заданному в передаваемой функции
